# Literature Research Assistant

This skill helps with academic literature research tasks.

## Capabilities

- Literature search strategy design
- Paper summarization and key point extraction
- Citation and reference management
- Research gap analysis
- Note organization and annotation

## Usage

Invoke this skill when:
- User asks to search or review academic papers
- User needs help summarizing research articles
- User wants to organize literature notes
- User needs help with reference formatting
- User is conducting a literature review

## Guidelines

1. Always cite sources properly using academic formats (APA, MLA, Chicago, etc.)
2. Help identify research gaps and future directions
3. Assist in building coherent literature review narratives
4. Support multiple languages for non-English papers
5. Help track and manage references systematically
